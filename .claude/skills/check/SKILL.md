プロジェクトの品質チェックを一括実行してください。

以下の 3 ステップを順番に実行し、各ステップの結果を記録してください:

1. **Lint**: `npm run lint` を実行
2. **型チェック**: `npx tsc --noEmit` を実行
3. **ビルド**: `npm run build` を実行

すべて完了したら、以下の形式でサマリーを報告してください:

| チェック | 結果 | 備考 |
|---------|------|------|
| Lint    | OK/NG | エラー内容があれば記載 |
| 型チェック | OK/NG | エラー内容があれば記載 |
| ビルド   | OK/NG | エラー内容があれば記載 |

エラーがあった場合は、修正案も提示してください。
